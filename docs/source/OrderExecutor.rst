基本使用指令
------------

这里放几个基本的指令

- ``.r``  --最基础的骰点
- ``.help``  --查看帮助(strHlpMsg)
- ``.help [词条名]``  -- 查看对应帮助文档
- ``.bot on/off``  --开关秩灵
.. note::

   | .sn和常见豹骰那些是不一样的，直接发送.sn即可，后续会持续更新保持指令一致
   | .reply off才能关闭自定义回复，bot off是关不上的

- ``.dismiss``  --请秩灵出群
.. DANGER::
   请注意，把秩灵踢出群会被拉黑，要解散群聊之前也请把秩灵提前使用.dismiss请出

- ``.draw``  --抽牌，点击待定查看装载牌堆
| 其余请查看 `用户手册 <https://v2docs.kokona.tech/zh/latest/User_Manual.html#>`_
